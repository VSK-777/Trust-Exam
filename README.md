# 🧪 Trust-Exam

An interactive **Online Examination Interface** built using HTML.  
This repository currently contains the frontend UI pages that simulate an online exam workflow.

---

## 📌 Project Overview

Trust-Exam is a prototype web-based examination system interface.  
It demonstrates the structure and navigation flow of an online exam platform.

This project currently focuses on:

✔ Landing Page  
✔ Exam Instructions Page  
✔ Exam Interface  
✔ Dashboard View  
✔ Result Page  

It is designed to be extended later with backend logic, authentication, database connectivity, and AI-based proctoring features.

---

## 🏗️ Tech Stack (Current Version)

- HTML5  
- Basic Web Structure  
- Static Page Navigation

## 🚀 How to Run the Project

1️⃣ Clone the repository:

```
git clone https://github.com/VSK-777/Trust-Exam.git
```

2️⃣ Open the project folder.

3️⃣ Double-click `index.html` to start in your browser.

No server setup required.  
This is a static frontend project.
